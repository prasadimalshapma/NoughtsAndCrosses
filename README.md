# NoughtsAndCrosses(Tic-Tac-Toe)
Tic-Tac-Toe (Noughts and Crosses) game in Python with human vs computer, score saving, and leaderboard.

Features:
- Human vs Computer Tic-Tac-Toe
- Score saving and leaderboard (`leaderboard.txt`)
- Menu system: Play, Save Score, Load Scores, Quit

How to Run:
python play_game.py
